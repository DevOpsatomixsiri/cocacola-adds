# Coca Cola Adds Website

A website project for Coca Cola advertisements and promotional content.

## Description

This repository contains the HTML and web assets for a Coca Cola adds (advertisements) website.

## Project Structure

```
cocacola-adds/
├── README.md
└── [HTML files and assets]
```

## Technologies Used

- HTML

## Getting Started

### Prerequisites

- A modern web browser
- A text editor for development (optional)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/DevOpsatomixsiri/cocacola-adds.git
```

2. Navigate to the project directory:
```bash
cd cocacola-adds
```

3. Open the HTML files in your web browser or serve them using a local web server.

## Usage

Open any `.html` file in your web browser to view the Coca Cola advertisement content.
